## noteへの投稿方法

### 方法1: 手動コピー&ペースト（推奨）

1. 各Markdownファイルを開く
2. 内容をコピー
3. noteの新規投稿画面に貼り付け
4. タイトル、カテゴリ、タグを設定
5. 公開

### 方法2: 一括インポート

1. _index.csvをExcelまたはGoogleスプレッドシートで開く
2. 記事ごとに順次noteに投稿
3. Markdownファイルの内容をコピー&ペースト
